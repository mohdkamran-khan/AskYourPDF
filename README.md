A small AskYourPDF app where users upload a PDF and ask questions. This repository is a minimal, demonstrable project that satisfies the Kaggle/Google Capstone submission requirements: an AI agent that accepts documents and answers questions.

## How to run
1. Open `server/.env` and add your OpenAI API key.
2. `cd server && npm install`
3. `cd ../client && npm install`
4. Start server: `cd server && npm run dev`
5. Start client: `cd client && npm run dev`
6. Open http://localhost:5173 and test.
